# SpellingBeePlus agent notes

These directories contain the application files:

- `server/` - backend server; this is where most application logic lives, split between client and server code
- `extension/` - Safari extension; this is mostly just a shell to run the downloaded application logic in Safari

These are for development only; ignore them unless instructed to update them:

- `sb-cli/` - development CLI tool for interacting with Spelling Bee API
- `sbp-cli/` - development CLI tool for interacting with SpellingBeePlus API
